# Trading Cheat Sheet

Особистий довідник торгових термінів (SMC / ICT / індикатори) з пошуком, обраним, світлою/темною темою і друком у PDF.

**Живий сайт:** `https://<username>.github.io/<repo>/` (заповнити після деплою)

## Як редагувати терміни

Усі дані — у файлі [`terms.json`](terms.json). HTML чіпати не потрібно.

### Через GitHub (з будь-якого пристрою)

1. Відкрити `terms.json` у репозиторії.
2. Натиснути олівець (✏️ Edit this file).
3. Знайти потрібну категорію → змінити/додати терміни.
4. `Commit changes` → через ~30–60 сек сайт оновиться.

### Схема одного терміна

```json
{
  "name": "HH",
  "abbr": "Higher High",
  "desc": "Вищий максимум. Кожен новий пік вище попереднього.",
  "use": "Ознака висхідного тренду (uptrend).",
  "keywords": "HH higher high higher highs хай хаї"
}
```

- `name` — основна назва (обов'язково).
- `abbr` — розшифровка в дужках (опційно).
- `desc` — короткий опис (обов'язково).
- `use` — "Як використовувати" (опційно).
- `keywords` — пробіл-розділений список слів/абревіатур для пошуку (укр + eng).

### Додати нову категорію

У масиві `"categories"` додати об'єкт:

```json
{
  "icon": "🔥",
  "title": "Назва категорії",
  "dataCategory": "Коротка назва",
  "terms": []
}
```

## Деплой на GitHub Pages

1. Створити публічний репозиторій, залити `index.html`, `terms.json`, `.nojekyll` (і цей README).
2. **Settings → Pages → Source: Deploy from a branch → `main` / root → Save**.
3. Через ~1 хвилину сайт доступний за `https://<username>.github.io/<repo>/`.

## Локальний запуск

```bash
cd TAsheet
python3 -m http.server 8000
# Відкрити http://localhost:8000
```

Відкривати `index.html` напряму через `file://` **не можна** — `fetch('terms.json')` вимагає HTTP.
